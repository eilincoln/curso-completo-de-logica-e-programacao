Dados

Dado é um termo genérico empregado para denotar quaisquer ou todos os números, letras e símbolos que se referem a, ou descrever um objeto, idéia, condição ou outros fatores. O termo indica de maneira indireta os elementos básicos de informação que podem ser processados ou produzidos por um computador.

  • São elementos conhecidos de um problema em questão.
  • É um conjunto de "informação em bruto" que, através de determinados processos, se transforma em informação para o usuário.

Cada caractere, letra ou símbolos vai gerar esses dados, e esses dados de forma organizada, vão gerar informação.

Os dados organização são informações.

Informação

  • Conjunto estruturado de dados, transmitindo conhecimento.
  • Conjunto de resultados que são obtidos após um processamento.
